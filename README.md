# console-calculator
**콘솔 기반 계산기 프로젝트**



## 프로젝트 개요

자바로 구현한 단계별 계산기 프로젝트입니다.  
Lv1부터 Lv3까지 기능을 점차 확장하며, 객체지향 설계와 콘솔 입출력, 컬렉션 활용, 예외 처리 등을 연습합니다.



## 기능 구성

| 레벨 | 기능 |
|------|------|
| **Lv1** | 두 개의 숫자와 사칙연산 기호(`+ - * /`) 입력<br> `exit` 입력 시 종료 |
| **Lv2** | 연산 결과를 컬렉션에 저장<br> 가장 먼저 저장된 결과를 삭제 |
| **Lv3** | 실수(소수점) 입력 허용<br> 특정 값보다 큰 결과만 조회 가능 |



## 사용 기술
Java 17+

콘솔 입출력

조건문 / 반복문 / 예외 처리

List 등 컬렉션 활용

패키지 구조 관리
